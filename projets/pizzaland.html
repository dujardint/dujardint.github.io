Pizzaland API

Nous avons crée une API REST permettant de récuperer tous les ingrédients, les pizzas et les commandes. L'ajout et la modification peut se faire uniquement identifié via l'obtention d'un token. Fonctionne dans un serveur tel que Apache Tomcat sous servlet java.

Objectif pédagogique :
Trouver une solution informatique permettant à un autre logiciel de communiquer avec un autre.
