    Mes projets

    Pendant mes années, j’ai accompli des projets scolaire et personnels. En voici une partie :

    Voici mon 1er projet de mon BUT, nous devions écrire le code pour une machine permettant d’encoder du texte utilisé pendant la guerre en langage java simplifié (ijava).

    Dans le cadre d’une politique de covoiturage, l'université nous avait demandé (en factif) de créer une plateforme de covoiturage afin de favoriser les déplacements en groupe au sein de l'université : TonCovoitUniv est crée !

    Dans le cadre d’une création d’entreprise par nous , nous devions en binôme créer notre entreprise. Pour cela, nous avons tout imaginé ; le nom, ses services, son implantation… L’entreprise DepInfo fût crée avec à sa tête 2 jeunes personnes fictives : Siana Lefebvre (Anaïs CASTEUR) et Marc Johns (Thomas DUJARDIN).

    Nous devions créer un logiciel ludo-pédagogique à destination d’élèves d’écoles primaires, avec une interface texte, Une seule contrainte était demandé : un aspect pédagogique. Le reste était pour nous. Nous avons pensé à 3 mini-jeux dans un jeu ! Quizzy est né !
